# Gasé

> Gasé is an ecommerce web app project for selling clothing products.

_Stack: HTML, CSS, Bootstrap, Django_

## Core features

-   Easy products navigation
-   Product search
-   Product tags
-   Cart management
-   Address and Billing manangement
-   Shoping for non regsitered user
